Chapter 6:Web Technology

Introduction to the Internet

The Internet is a global network of networks, interconnecting computers worldwide to form a web-net.

Components Required for Internet Connection

  • A TCP/IP-enabled computer with a web browser
  • An account with an ISP (Internet Service Provider)
  • A telephone line
  • A MODEM to connect to the telephone line and computer
  • A computer with minimum specifications:
    • 386 microprocessor chip with 16 MB RAM
    • Color monitor with 256 color support and 640x480 resolution
    • 200 MB free hard disk space
    • Multimedia sound card and speakers

Internet Services and Applications

  • World Wide Web (WWW): A series of services interconnected through hypertext, using HTTP protocol.
  • Web Browser: A program to access and display web pages (e.g., Chrome, Firefox, Safari).
  • FTP (File Transfer Protocol): Transfers files between computers.
  • Email: Sends and receives electronic messages.
  • Voicemail: A system for recording and storing voice messages.
  • USENET: A bulletin board service for news and discussions.
  • Chat: Real-time text-based communication.
  • IRC (Internet Relay Chat): Multi-user live chat service.
  • Video Conferencing: Live video communication over the Internet.
  • URL (Uniform Resource Locator): The unique address of a webpage.
  • E-Fax: Internet-based fax service.
  • TELNET: Remote login to another computer.
  • Navigators: Tools to locate and retrieve online information.

Internet Terminologies

  • ISP: Internet Service Provider (e.g., WorldLink, Mercantile).
  • Link: A hyperlink to another webpage.
  • Search Engine: A tool to search for information (e.g., Google, Yahoo).
  • Domain Name & IP Address: Unique identifiers for websites.
  • DNS Server: Translates domain names into IP addresses.
  • Web Server: Stores and delivers webpages.
  • Newsgroups: Online discussion forums.
  • Mailing Lists: Groups for email-based discussions.
  • POP Server: Stores incoming emails.
  • SMTP Server: Handles outgoing emails.
  • FTP: Transfers files between systems.
  • Anonymous FTP: Accesses files without authentication.
  • HTTP: Protocol for web communication.
  • Proxy Server: Improves browsing speed and security.
  • Internet Roaming: Allows using an Internet account worldwide.

Q1. Define HTML. Explain its features. [5 Marks]

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures web content using elements and tags.

Features of HTML:

Feature Description
Platform Independent HTML works on all operating systems and browsers.
Simple & Easy Easy to learn and use for website development.
Supports Multimedia Can integrate images, audio, video, and animations.
Hyperlinks Allows linking to other web pages and resources.
Flexible & Extensible Works with CSS and JavaScript for advanced designs.

Q2. Define tags and their types with examples. [5 Marks]

HTML Tags: Tags are predefined keywords enclosed in angle brackets (< >) that define the structure of web content.

Types of Tags:

Type Description Example
Paired Tags Have an opening and closing tag. <p>This is a paragraph</p>
Self-closing Tags Do not require a closing tag. <img src="image.jpg" alt="Image">

Q3. Define DBMS. Explain its advantages. [5 Marks]

DBMS (Database Management System) is software that manages databases, allowing users to store, retrieve, and manipulate data efficiently.

Advantages of DBMS:

Advantage Description
Data Security Provides access control to protect data.
Data Integrity Maintains accuracy and consistency of data.
Easy Data Retrieval Allows quick access to stored information.
Data Redundancy Reduction Minimizes duplicate data storage.
Multi-user Access Supports concurrent data access by multiple users.

Q4. Important HTML Tags and Attributes for MCQs

1. Character Formatting Tags:

  • <b>Bold</b>
  • <i>Italic</i>
  • <u>Underline</u>

2. Attributes of Body Tag:

  • bgcolor: Sets background color (e.g., <body bgcolor="lightblue">).
  • text: Sets text color.
  • link: Sets link color.

3. Marquee Tag Attributes:

  • behavior: Defines movement type (scroll, slide, alternate).
  • direction: Specifies movement direction (left, right, up, down).

4. Image Tag Attributes:

  • src: Specifies image location.
  • alt: Alternative text for image.
  • width & height: Defines image dimensions.

5. Table Tag Attributes:

  • border: Defines border size.
  • cellpadding: Space inside each cell.
  • cellspacing: Space between table cells.

6. Ordered and Unordered Lists:

  • <ul>: Creates an unordered (bulleted) list.
  • <ol>: Creates an ordered (numbered) list.
  • <li>: Defines each list item.
