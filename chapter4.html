Chapter 4:Application Package

MS-Word (Word Processor)

MS-Word is a text editing or word processing software developed by Microsoft Corporation, USA, as part of the MS-Office package.

Features of MS-Word

  • Fast Operation: No mechanical movements, making it faster.
  • Editing Text Features: Insert, delete, modify, undo/redo, find/replace, and customize text (bold, italic, underline, font size, color, style, etc.).
  • Permanent Storage: Documents can be saved and retrieved from storage media.
  • Graphics: Allows inserting pictures, charts, and graphs.
  • OLE (Object Linking and Embedding): Enables sharing of objects like charts, equations, images, audio, and video.
  • Mail Merge: Automates creating multiple letters, labels, and envelopes using stored data.
  • Searching: Find and replace text throughout the document.

MS-Excel (Spreadsheet)

MS-Excel is a spreadsheet software in the MS-Office suite that allows users to store, organize, and analyze numerical and text data.

Application Areas of Excel

  • Basic mathematical calculations
  • Financial modeling and analysis
  • Statistical analysis
  • Scientific and engineering calculations
  • Graphic presentations

File Extensions: The file extension of MS-Excel is ".xlsx" and ".xls".

Common Terminologies in Spreadsheet

  • Worksheet: A collection of rows and columns used for organizing and calculating data.
  • Workbook: A file that contains multiple worksheets. By default, a workbook has three worksheets.
  • Cell: The intersection of a row and a column (e.g., A1, B3, C5).

Features of MS-Excel

  • Supports multiple worksheets.
  • Allows creation of different charts and graphs.
  • Manages large volumes of data.
  • Performs automatic calculations using formulas and functions.
  • Sorts data in ascending and descending order.
  • Supports Object Linking and Embedding (OLE).
  • Provides formatting tools for text and numbers.
  • Includes basic computing operations like cut, copy, paste, find, and replace.
  • Offers web features such as online help and web linking.
  • Stores data electronically in any storage medium.

Cell Reference in Excel

A cell reference is the address of a cell in a spreadsheet. It combines a column letter and a row number (e.g., A1, B3, Z9).

Types of Cell References

  • Relative Cell Reference: Changes when copied to another location (e.g., A1, B4, G9).
  • Absolute Cell Reference: Remains fixed using a dollar sign (e.g., $A$2, $G$9, $Z$345). Use the "F4" key to add the dollar sign.

MS-PowerPoint (Presentation)

PowerPoint is a presentation software in the Microsoft Office suite that allows users to create dynamic and professional presentations using pre-defined layouts, themes, and templates.

File Extensions:

The file extension of PowerPoint is ".ppt" (older versions) and ".pptx" (newer versions). Alternative software includes MagicPoint, Apple Keynote, IBM Lotus, and Harvard Graphics.

Features of MS-PowerPoint

  • Allows formatting of text using font styles, colors, and hyperlinks.
  • Supports basic operations like cut, copy, paste, find, and replace.
  • Provides pre-defined layouts, themes, and templates for quick presentation design.
  • Includes slide transition effects for smoother presentations.
  • Enables custom animations.
  • Supports internal and external hyperlinks.
  • Allows adding charts, graphs, and tables for better data visualization.
  • Includes web-related features like online help and web linking.

3. Define Cell Referencing. Explain Its Types. [5]

Cell referencing refers to the method of identifying a cell or a range of cells in a spreadsheet for use in formulas and functions.

Types of Cell Referencing:
  • Relative Referencing: The reference changes when copied to another cell (e.g., A1 becomes B1 when copied to the right).
  • Absolute Referencing: The reference remains fixed, denoted by "$" (e.g., $A$1 stays the same when copied).
  • Mixed Referencing: Either the row or column remains fixed while the other changes (e.g., $A1 or A$1).
